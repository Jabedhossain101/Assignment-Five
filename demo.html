Question-1: What are the different ways to select an element in the DOM?

Ans: getElementById(id): Selects an element by its id.
1.getElementsByClassName(class): Selects elements by class name.
2.getElementsByTagName(tag): Selects elements by tag name.
3.querySelector(selector): Selects the first matching element using CSS selectors.
4.querySelectorAll(selector): Selects all matching elements using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.innerHTML: Returns or sets the HTML content inside an element.
2.innerText: Returns or sets only the visible text, ignoring hidden elements.
3.textContent: Returns or sets all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

A technique where a single event listener is attached to a parent element to handle events on its child elements using event bubbling

Question-4: What is event bubbling in the DOM?

A process where an event triggered on a child element propagates up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

1.document.createElement(tag): Creates a new HTML element.
2.appendChild(element): Adds an element as a child to another element.
3.insertBefore(newElement, referenceElement): Inserts an element before another.
4.removeChild(element): Removes a child element from its parent.
5.element.remove(): Directly removes an element from the DOM.
